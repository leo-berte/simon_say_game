# simon_says_game

Try to build this simple but very funny game by yourself.
If you want to move your first steps into the amazing world of engineering, this project is made for you!

===============================

Components :

- 1 Arduino Pro Mini
- 4 Leds
- 4 Switches
- 1 Buzzer
